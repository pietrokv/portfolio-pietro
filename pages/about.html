Sobre Mim

Sou estudante em transição de carreira para a área de tecnologia, com foco em desenvolvimento web front-end. Venho me capacitando através de cursos de HTML, CSS e JavaScript na plataforma de estudos da Alura, com prática diária de escrita e organização de código no VS Code. Desenvolvi meu portfólio pessoal como projeto de estudo, estruturando páginas com HTML semântico, estilização com CSS (incluindo variáveis de tema e Flexbox para layout responsivo) e versionamento com Git/GitHub. O projeto foi essencial para consolidar minha lógica de programação e desenvolver autonomia na resolução de problemas técnicos. No momento, meu objetivo é aprofundar meus conhecimentos em JavaScript e boas práticas de front-end, com a meta de desenvolver sites modernos e funcionais para negócios reais — incluindo dois projetos da minha família que pretendo colocar no ar em breve. Tecnologias em desenvolvimento: HTML, CSS, JavaScript, Git/GitHub, VS Code. Características pessoais: comunicativo, proativo, com bom trabalho em equipe e forte vontade de aprender continuamente.
